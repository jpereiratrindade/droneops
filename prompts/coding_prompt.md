# Coding Prompt

Ao trabalhar no DroneOps:

- Use o MorfoCampo como referencia tecnica e visual.
- Mantenha o SisTer-Campo como fonte de arquitetura e governanca.
- Trate CampoSync como parte do escopo inicial, nao como recurso futuro.
- Nao implemente linguagem que sugira autorizacao automatica de voo.
- Preserve operacao offline-first.
- Adicione testes para regras de status, checklist, manifestos e exportacao.
