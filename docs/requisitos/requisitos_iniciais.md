# Requisitos Iniciais

## DroneOps

| ID | Requisito |
| --- | --- |
| RF-DO-01 | Cadastrar missao vinculada a projeto, campanha e area de operacao. |
| RF-DO-02 | Registrar equipe, funcao, aeronave, sensor, baterias e equipamentos auxiliares. |
| RF-DO-03 | Registrar planejamento, objetivo, data, horario e condicoes ambientais. |
| RF-DO-04 | Executar checklist pre-voo e checklist pos-voo. |
| RF-DO-05 | Anexar documentos, autorizacoes, mapas, plano de voo, imagens, logs e relatorios. |
| RF-DO-06 | Registrar ocorrencia, abortagem, falha, risco observado ou inconsistencia operacional. |
| RF-DO-07 | Calcular status operacional da missao. |
| RF-DO-08 | Gerar pacote de missao compativel com CampoSync. |

## CampoSync

| ID | Requisito |
| --- | --- |
| RF-CS-01 | Gerar manifesto para pacote de missao. |
| RF-CS-02 | Calcular hash dos arquivos exportados. |
| RF-CS-03 | Validar presenca de evidencias referenciadas. |
| RF-CS-04 | Registrar status de exportacao e sincronizacao. |
| RF-CS-05 | Preparar pacote para importacao manual ou assistida no SisTer. |
